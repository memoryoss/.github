# MemoryOSS

**Community-driven, research-backed AI memory infrastructure — like Kubernetes, but for memory.**

`memoryoss` is an open-source organization building **AI memory solutions for the next generation of intelligent systems.**  
Our goal is to create a **neutral, community-owned foundation**.

---

## What we do

- Build **open, extensible memory infrastructure**
- Enable **self-hosted, production-ready deployments**
- Support **bring-your-own LLM, database, and stack**
- Follow a **zero-external-dependency design principle**

---

## Projects

### MemWire
An **enterprise-ready AI memory infrastructure layer**

- Unified memory (conversations + knowledge)  
- Fast semantic retrieval  
- Automatic memory classification (no external LLM)  
- Python SDK + REST API  
- Fully deployable anywhere
